Project Plan

Task Manager Application

A planned web application for managing tasks, roles, statuses, priorities, and workflow progress.

01

Project Idea

The main idea of this project is to create a task management system that helps users organize work, assign tasks, track progress, and manage priorities. The application is planned as a practical student project with a real project structure.

Main Features

  • Adding new tasks
  • Task statuses: To Do, In Progress, Done
  • Task priorities: Low, Medium, High
  • Search and filtering
  • Saving data in a database

User Roles

  • Admin can manage all tasks
  • Manager can create and assign tasks
  • User can update personal task progress

Frontend

  • HTML for page structure
  • CSS for modern responsive design
  • JavaScript for interactions and dynamic task logic

Backend & Database

  • Node.js for server-side logic
  • Express.js for API routes
  • SQL database for storing users and tasks
  • REST API for frontend/backend communication
02

Planned Database Structure

Users table

id, name, email, password, role

Tasks table

id, title, description, status, priority, deadline, user_id

03

Development Plan

Step 1

Create layout and basic interface with HTML and CSS.

Step 2

Add JavaScript logic: creating tasks, filtering, changing status.

Step 3

Create Node.js backend with Express routes.

Step 4

Connect SQL database and save task information.

Step 5

Test the application and prepare final presentation.
